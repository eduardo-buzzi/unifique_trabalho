# 🌐 Projeto Unifique - Site Institucional + Plataforma de Cursos

Projeto desenvolvido como atividade acadêmica para o curso de **Análise e Desenvolvimento de Sistemas (UniSenai)**.

O sistema simula o site de uma provedora de internet (**Unifique**), incluindo:
- Página institucional
- Página de cursos de TI
- Página de planos/produtos

---

## 🚀 Tecnologias Utilizadas

- HTML5
- CSS3 (Grid + Responsividade)
- Design moderno com foco em UI/UX

---

## 💻 Funcionalidades

- ✅ Layout responsivo (mobile, tablet e desktop)
- ✅ Navegação entre páginas
- ✅ Catálogo de cursos
- ✅ Página de produto com benefícios e depoimentos
- ✅ UI moderna com CSS Grid
